Language Translator

  • Tech Stack: Django, HTML, CSS, Bootstrap, REST API, JavaScript
  • Github URL: Project Link

Django based language translating website.

The objective of this project is to provide an application to the people which will resolve the problem of communication gap between the people due to lack of languageâ€™s knowledge. This project aims to recognise voice, to translate the recognised text from one language to another and to read the translated text.

It assumes that the input is in the same language as per selected in the selection box. Any word in input which is not found in a standard dictionary may result in inaccurate tags. For now, Language-Translator only supports 10 languages : 1. English 2. Hindi 3. French 4. German 5. Japanese 6. Korean 7. Russian 8. Turkish 9. Tamil 10. Spanish The application is not providing option for other languages but in future there will be more options in the same.
